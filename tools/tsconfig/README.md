# npm
```sh
npm install -D @gratisvictory/tsconfig
```
# yarn
```sh
yarn add -D @gratisvictory/tsconfig
```
# pnpm
```sh
pnpm add -D @gratisvictory/tsconfig 
```
# configuration file
```js
// tsconfig.json
options: tsconfig.react.json | tsconfig.next.json

{
  "extends": ["@gratisvictory/tsconfig/[options]"],
}
```
