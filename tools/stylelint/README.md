# npm
```sh
npm install -D @gratisvictory/stylelint
```
# yarn
```sh
yarn add -D @gratisvictory/stylelint
```
# pnpm
```sh
pnpm add -D @gratisvictory/stylelint 
```
# configuration file
```js
// .stylelint.cjs
const { stylelint } = require('@gratisvictory/stylelint')

module.exports = stylelint

```
